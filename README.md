# ANN_DIABETES_ANALYZER
# 🩺 ANN_DIAB — Diabetes Prediction using Artificial Neural Networks

A professional deep learning project for predicting diabetes using an Artificial Neural Network (ANN) built with TensorFlow/Keras and deployed with Streamlit.

---

## 📌 Project Overview

ANN_DIAB is a machine learning project focused on binary classification for diabetes prediction. The model analyzes patient medical attributes and predicts whether a person is likely to have diabetes.

The trained model is stored in Keras format (`.keras`) and integrated with a Streamlit web application for real-time predictions.

## 📊 Input Features

| Feature | Description |
|---|---|
| Pregnancies | Number of pregnancies |
| Glucose | Plasma glucose concentration |
| Blood Pressure | Diastolic blood pressure |
| Skin Thickness | Triceps skin fold thickness |
| Insulin | Serum insulin level |
| BMI | Body Mass Index |
| Diabetes Pedigree Function | Diabetes inheritance score |
| Age | Age of patient |

---

## 📈 Output

The model predicts:

```text
0 → Non-Diabetic
1 → Diabetic
```
